# AppDB-Scraper
Scrape download links from AppDB

# Installation
```
git clone https://github.com/PixelMelt/AppDB-Scraper/
cd AppDB-Scraper/src
npm i axios
```

# Usage
```
node appdbdownloader.js <LINK TO APPDB APP>
```
